# Простая TDS

Небольшая система распределения трафика на Flask. Подойдёт даже для слабых серверов.

## Возможности

- Авторизация администратора (по умолчанию `admin`/`admin`).
- Создание, редактирование и удаление кампаний.
- Ротация между несколькими URL в рамках одной кампании.
- Запасной URL при срабатывании гео‑фильтра.
- Ограничение по странам.
- Несколько видов редиректов: `302`, meta refresh, JS, заголовок `Refresh`, `iFrame`.
- Пользовательский путь (slug) для коротких ссылок.
- Учёт кликов с IP, страной и user‑agent.
- Подстановка макросов `{ip}`, `{country}`, `{ua}` в целевых ссылках.

## Запуск

```bash
pip install -r requirements.txt
python app.py
```

Приложение слушает порт 5000. При обновлении версии удалите файл `tds.db`, чтобы база создалась заново.

Переменная окружения `SECRET_KEY` меняет секретный ключ сессий.

## Использование

1. Перейдите на `http://localhost:5000` и авторизуйтесь.
2. Создайте кампанию в форме сверху.
3. Введите название, при желании свой slug, основной и дополнительные URL.
4. Укажите страны через запятую или оставьте поле пустым.
5. Выберите тип редиректа и задержку.
6. Скопируйте ссылку вида `/t/ID` или `/s/slug` из таблицы и распространяйте её.
7. В разделе «Статистика» можно посмотреть собранные клики.
